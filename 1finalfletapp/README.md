# WebSec Scanner 🛡️

WebSec Scanner is a comprehensive, passive web security auditing suite. It features a centralized API backend that powers multiple frontend interfaces, allowing you to scan targets for SQLi, XSS, and Sensitive Data Exposure via a desktop app or directly from your web browser.

## 🏗️ System Architecture

The project is modular and consists of the following components:

* **Backend (`scanner.py` & `database.py`):** A FastAPI server that handles the core scanning engine and rate-limiting. It uses a pure SQLite database (`websec.db`) to store logs, API keys, and admin credentials.
* **Desktop Frontend (`app2.py`):** A sleek, dark-themed local application built with Flet.
* **Browser Extension (`websec-extension/`):** A lightweight Chrome/Edge extension that mirrors the desktop app's functionality, allowing you to scan the current tab with a single click.
* **Admin Dashboard (`admindashboard.py`):** A Streamlit-based global dashboard for viewing all network scans, system logs, and security metrics.

---

## ⚙️ Configuration & API Keys

Access to the scanning engine is protected by API keys.

1.  **Initialization:** When the backend runs for the first time, `database.py` automatically initializes the database, creates a default admin user, and generates a bootstrap API key. Check your terminal output to copy this bootstrap key.
2.  **Hardcoded Defaults:** By default, both the Flet app and the browser extension have a local API key hardcoded for convenience during development. 
3.  **Override:** If the hardcoded API key is revoked or does not work, simply copy the bootstrap key generated by `database.py` (or generate a new one via the backend) and paste it into the API key field in either frontend application.

## 🚀 Getting Started

### 1. Install Dependencies
Run the provided batch script to install all required libraries:
```bash
install_requirements.bat
```
*(Alternatively, run `pip install -r requirements.txt` manually).*

### 2. Start the Backend API
The FastAPI backend must be running on port 8000 before you launch any frontends.
```bash
uvicorn scanner:app --host 0.0.0.0 --port 8000
```

### 3. Start the Admin Dashboard
Open a new terminal and run the Streamlit dashboard:
```bash
streamlit run admindashboard.py
```
**Admin Login Credentials:**
* **Username:** `admin`
* **Password:** `changeme123!`
*(Ensure you change these in a production environment!)*

## 💻 Launching the Frontends

You can use either the Desktop App or the Browser Extension to run your scans.

### Option A: Desktop App (Flet)
Open a new terminal window and run:
```bash
python app2.py
```

### Option B: Browser Extension
To install the extension in Google Chrome, Microsoft Edge, or Brave:
1. Open your browser and navigate to the extensions page (e.g., `chrome://extensions/` or `edge://extensions/`).
2. Enable **Developer mode** (usually a toggle in the top right corner).
3. Click the **Load unpacked** button.
4. Select the `websec-extension` folder from this repository.
5. Pin the WebSec shield icon 🛡️ to your browser toolbar.
6. Click the extension, enter your target URL (or click "use current tab"), ensure your API key is pasted in, and start scanning!
